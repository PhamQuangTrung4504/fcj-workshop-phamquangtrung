### Chủ đề tuần

Tối ưu hóa chi phí với Lambda + API Gateway + CloudFormation

### Mục tiêu tuần

* Hiểu cách AWS Lambda hỗ trợ tự động hóa vận hành EC2 và tối ưu hóa chi phí.
* Tìm hiểu cách thiết kế và cung cấp các REST API với Amazon API Gateway.
* Bắt đầu với AWS CloudFormation để quản lý cơ sở hạ tầng dưới dạng mã (Infrastructure as Code).

### Kết quả mong đợi

* Hiểu vai trò của Lambda trong tự động hóa và kiểm soát chi phí EC2.
* Có khả năng thiết kế và cung cấp các REST API cơ bản sử dụng API Gateway.
* Tích lũy kinh nghiệm ban đầu trong việc viết và triển khai các template CloudFormation.
